### 3.2.3 检索逻辑

```python
from sklearn.metrics.pairwise import cosine_similarity
import numpy as np

def search_similar_content(self, query: str, n_results: int = 5):
    # 1. 将查询向量化
    query_embedding = self.model.encode([query])

    # 2. 计算与所有文本块的余弦相似度
    similarities = cosine_similarity(query_embedding, self.embeddings)[0]

    # 3. 取Top-K
    top_indices = np.argsort(similarities)[-n_results:][::-1]

    return {
        'documents': [self.documents[i] for i in top_indices],
        'metadatas': [self.metadata[i] for i in top_indices],
        'similarities': [similarities[i] for i in top_indices]
    }
```

```json
{
  "name": "PDF知识库问答分析",
  "version": "1.0",
  "parameters": {
    "temperature": 0.3,
    "top_p": 0.9,
    "max_tokens": 4096,
    "repeat_penalty": 1.1
  },
  "knowledge_base": {
    "path": "data/raw",
    "file_count": 35,
    "purpose": ["问答", "分析", "总结"]
  }
}
```

**使用方式**：将检索到的相关文本块作为上下文，注入 API Prompt：

```
系统：你是一个专业的敦煌藻井研究助手。基于以下文献内容回答问题。

用户：{检索到的文本块}

用户：{原始问题}
```

### 3.4.3 交互功能

- **朝代筛选**：点击按钮过滤时间线卡片（CSS `hidden` 类切换）
- **纹样筛选**：过滤纹样分类卡片（`display: none` 切换）
- **色块点击**：弹出矿物颜料说明（DOM 显示/隐藏）
- **语义检索**：关键词匹配 + 高亮（正则替换 + 相关度评分）
- **问答对话**：预设问题 + 模拟延迟回复 + 来源标注
